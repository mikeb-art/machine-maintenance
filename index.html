Filter / Head Change Log

Log every filter and head change on the machine’s Filter/Head Change Log (Google Sheet). Record:

  • Date, and filter or head changed (part + location, e.g. C1, YM4)
  • Circulation pressure — before / after
  • Circulation pump frequency — before / after (Durst normal: 4.5–6.5 pressure, 65–110 pump freq)
  • Printed sq ft (or sq m) at time of change
  • Ink consumption (L) for the color changed
  • Technician name + notes

P5-Tex parts: carriage filter JKP 5µm PP (Durst P/N 1392545) · main ink tank filter MKL 5µm PP BLK (P/N 1392994). Values are found under Ink Heater (pressure/frequency) and Ink Cost (output, consumption) — never press Reset.
